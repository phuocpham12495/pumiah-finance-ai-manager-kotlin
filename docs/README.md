# Pumiah Finance AI Manager — Android

Ứng dụng quản lý tài chính cá nhân thông minh cho Android, chuyển đổi từ phiên bản React Web.

## Thông tin dự án

| Mục | Chi tiết |
|-----|----------|
| Nền tảng | Android (API 26+) |
| Ngôn ngữ | Kotlin 2.1.0 |
| UI Framework | Jetpack Compose + Material Design 3 |
| Backend | Supabase (https://nlfsyjqkcgswdatpdaee.supabase.co) |
| AI | Google Gemini 2.5 Flash |
| Kiến trúc | MVVM + Repository + Hilt DI |
